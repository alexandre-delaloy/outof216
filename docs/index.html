node-typescript-starter

A NodeJS TypeScript starter, including TSLint, Jest, GitHub Actions & code coverage

Features available:

  • Linting (TSLint)
  • Testing (Jest)
  • Code coverage (Jest, need a secret token)
  • CI/CD (GitHub Actions)
  • Publishing your package to NPM (need a secret token)
  • Source code documentation generation and website (TypeDoc)

Purpose

*"The purpose of this starter is to create NodeJS executables, by making qualitative code, while automating publishing & CI/CD"*

Install

 
git clone git@github.com:blyndusk/node-typescript-starter.git
  • Set a GitHub Pages to ./docs folder.
  • Register your repo to Codecov.
  • Update the badges with your repo's name (and eventually secret token)

Usage

Starting

Lint source file while fixing warnings & errors, then build source files & test while watching.

npm run start

Linting & Building

 
npm run lint
npm run lint:fix

Convert .ts files to `.js, then add#!/usr/bin/env node` to the file to make it executable.

npm run build

Testing

Coverage available [here](https://blyndusk.github.io/ node-typescript-starter/lcov-report).

npm run test
npm run test:watch

Checking

Basically used before pushing, will lint while fixing, build & test.

npm run check

Documentation

Documentation available [here](https://blyndusk.github.io/ node-typescript-starter).

npm run doc

CI/CD
  • Go to Codecov
  • Login with Github
  • Register your repository
  • Copy the token and create a GitHub secret: CODECOV_TOKEN
  • Go to Shields.io to create a new Codecov badge with your token
